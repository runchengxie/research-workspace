样本覆盖 A 股日线和估值数据 `2008-01-02 ~ 2026-06-26`。因子收益从各因子可用后开始，主体覆盖

## 先读方向

这些因子是参考 Barra 思路构造的风格代理因子。每个因子都在月末按五分位分组，构造等权多空组合：

| 因子收益为正时 | 含义 |
| --- | --- |
| Size | 大市值强于小市值 |
| Value | 低 PB 强于高 PB |
| Momentum | 过去 21 日强势股继续强于弱势股 |
| Quality | 低 PE_TTM 强于高 PE_TTM；当前更接近盈利收益率 |
| LowVol | 低波动强于高波动 |
| Growth | 净利润/营收增长强于低增长 |
| Leverage | 低资产负债率强于高资产负债率 |
| Beta | 低 beta 强于高 beta |
| Liquidity | 低换手强于高换手 |

反过来，Size 为负表示小市值强，Momentum 为负表示短期反转强，LowVol 为负表示高波动强，
Beta 为负表示高 beta 强，Liquidity 为负表示高换手强。

## 长期画像

2008-2026 的全样本里，Growth、低换手和低估值是更稳定的正收益风格：

| 因子 | 年化收益 | 夏普 | 解读 |
| --- | ---: | ---: | --- |
| Growth | +8.45% | 1.24 | 成长基本面代理因子长期最稳 |
| Liquidity | +12.82% | 0.69 | 低换手/低拥挤股票长期补偿明显，但波动高 |
| Value | +7.20% | 0.44 | 低 PB 长期有效，但阶段性很强 |
| Leverage | +4.29% | 0.36 | 低杠杆长期有温和溢价 |
| LowVol | +4.33% | 0.24 | 防御风格长期略正，阶段波动较大 |
| Quality | +1.73% | 0.12 | 当前代理因子与 Value/LowVol 共线，独立性有限 |
| Beta | -7.77% | -0.40 | 当前定义为低 beta - 高 beta，结果显示高 beta 长期有阶段性优势 |
| Momentum | -10.94% | -0.83 | 21 日短动量长期更像反转，直接追短期强势的效果较差 |

需要特别小心共线性。全样本里 LowVol、Liquidity、Beta、Quality 之间相关性很高：
LowVol-Liquidity 约 `0.91`，LowVol-Beta 约 `0.82`，Beta-Liquidity 约 `0.81`，
Quality-Beta 约 `0.81`。这些因子经常同时描述低波、低换手、低 beta、低估值或高盈利收益率这一类
防御组合。解读时应把它们当成相近风格，避免逐个拆成独立因果。

## 风格切换脉络

### 1. 小盘长期强，阶段上会反复切换

Size 是全样本最弱因子，年化 `-19.71%`。由于 Size 定义为大市值 - 小市值，这代表小市值相对大市值
长期明显占优。2008-2016 大部分年份都是小盘强，2017 是少数非常明确的大盘强年份，2020 和 2026
也出现 Size 转正。

这点很重要：如果一个策略长期跑赢但没有做市值中性，它很可能吃到了小盘 beta。反过来，2017 或
2026 这类 Size 转正阶段，会明显惩罚过度小盘暴露。

### 2. 短动量长期更接近反转信号

Momentum 是过去 21 日强势 - 弱势。全样本年化 `-10.94%`，多数年份为负，说明 A 股在这个频率上
更偏短期反转。2026 年初至今的 `+32.6%` 是一个明显例外，代表短期强势股阶段性延续。

### 3. 低换手是长期有效但会被交易热度阶段性打断

Liquidity 为正代表低换手强。长期年化 `+12.82%`，并且 2008、2011、2016、2017、2018、2022 都很强。
但 2025 和 2026 年初至今为负，说明市场阶段性切到高换手、高交易热度方向。这个切换通常会让低波、
低 beta、低换手类防御风格一起受压。

### 4. 价值和成长轮流占优

Value 在 2009、2014、2016、2022、2023、2024 明显强；Growth 在 2010、2012、2013、2020、2025、
2026 年初至今明显强。两类风格随市场环境轮动。

较典型的切换是：

- 2020：Growth `+19.6%`，Value `-15.2%`，成长显著压过价值。
- 2022-2024：Value 连续为正，尤其 2022 `+27.1%`、2023 `+23.9%`、2024 `+17.5%`。
- 2025-2026 年初至今：Growth 重新为正，Value 转弱，2026 年初至今 Value 为 `-25.3%`。

### 5. 防御风格常常抱团出现

LowVol、Liquidity、Beta、Quality 在相关性上高度绑定。2017、2021、2022、2024 这类年份，低波、
低换手、低估值/质量往往一起强。2026 年初至今整体转向：LowVol `-27.2%`、Liquidity `-22.1%`、
Beta `-24.1%`，说明低波/低换手/低 beta 这组防御属性整体被压制。

## 2026 年初至今：历史对比

2026 年初至今的风格向量是：

```text
Size +23.8, Value -25.3, Momentum +32.6, Quality -16.3,
LowVol -27.2, Growth +16.4, Leverage -0.2, Beta -24.1, Liquidity -22.1
```

直接解读是：

- 大市值强于小市值，和 2017、2020 类似；2008-2016、2021-2025 多数阶段更偏小盘占优。
- 短动量显著转正，和多数历史年份不同；历史上短动量长期偏反转，2026 年初至今更像短期趋势延续。
- 低波、低换手、低 beta 同时为负，说明更活跃、更高波动、更高 beta 的组合阶段性占优。
- Growth 为正、Value 为负，接近 2020 和 2025 的成长相对价值格局。
- Leverage 接近 0，说明低杠杆和高杠杆对 2026 年初至今的解释力有限。

从跨因子年度形态看，2026 年初至今与 2020 最相似，其次接近 2025；和 2016、2022、2008、2011 这类
低换手/低波/价值防御年份差异很大。更直接地说，2026 年初至今呈现出
成长相对价值转强、短动量延续、大市值阶段性占优、防御风格承压的组合。

## 对策略解读的含义

如果某条策略在 2026 年初至今表现好，需要先检查它是否暴露在这些方向上：

- 是否偏大市值；
- 是否追逐短期强势或近期涨幅；
- 是否偏成长，低 PB 暴露是否较低；
- 是否回避低波/低换手防御股；
- 是否不再受益于过去几年常见的价值、低波、低换手组合。

如果策略在 2022-2024 好、但 2026 年初至今弱，一个常见解释是它吃到过 Value/LowVol/Liquidity 防御 beta，
而 2026 这组风格反向。如果策略在 2026 年初至今好，也需要先排查风格 beta；它可能只是顺着 Momentum、
Growth 和 Size 转正的风格切换。需要接入策略日收益 CSV，生成 `strategy_attribution_yearly.csv` 后，
才能把策略收益拆成风格 beta 和 residual alpha。

## 使用边界

这份解读只基于当前 9 个代理因子的多空收益，暂未包含行业中性化、交易成本、涨跌停成交约束、ST 过滤、
真实组合持仓和风险模型协方差。它适合回答市场奖励了什么风格，组合归因和风控归因仍需单独完成。
对于具体策略，下一步应使用策略日收益运行 OLS 归因，再结合持仓暴露、行业暴露和换手成本复核。
